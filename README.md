# Проект Auth

# Описание проекта
Web-приложение для использования RestFul API архитектуры

# Стек технологий
- Сборка приложения в Maven
- Java 17
- Spring Boot
- Spring Web
- Spring Data jpa
- Lombok

# Требования к окружению
- Java 17
- Maven 3.8
- PostgreSQL 14

# Запуск проекта
- Создать БД create database fullstack_auth
- Запустить проект по команде mvn spring-boot:run
